# ⌨️ Typing Speed Test

A real-time typing speed test built with HTML, CSS, and JavaScript.

## Features
- Live WPM and accuracy calculation as you type
- Character-by-character green/red highlighting
- 30-second countdown timer with auto-start on first keystroke
- 3 difficulty levels — Easy, Medium, Hard
- Final score screen with WPM, accuracy, and error count

## Tech Stack
HTML | CSS | JavaScript
